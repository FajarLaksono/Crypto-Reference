# Crypto Reference

## IDEA Cipher

IDEA (International Data Encryption Algorithm) awalnya disebut IPES (Improved proposed Encryption Standard).

* Pencipta:
    - Xuejia Lai
    - James Massey
* Kategori: Block Cipher
* Ukuran block (bit): 64
* Ukuran kunci (bit): 128
* Jumlah round: 8.5
* Struktur: Lai-Massey Scheme
* Diturunkan dari:
    - PES
* Penerus:
    - MMB
    - MESH
    - [Akelarre](../Akelarre)
    - [IDEA-NXT / FOX](../IDEA-NXT)
