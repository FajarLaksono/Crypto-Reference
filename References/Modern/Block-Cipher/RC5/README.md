# Crypto Reference

## RC5 Cipher

* Pencipta: Ron RIvest
* Kategori: Block Cipher
* Ukuran block (bit): 32 / 64 / 128 (disarankan 64)
* Ukuran kunci (bit): 0 - 2040 (disarankan 128)
* Jumlah round: 1 - 255 (disarankan 12)
* Struktur: Feistel-like Network
