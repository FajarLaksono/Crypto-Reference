# Crypto Reference

## DES Cipher

Data Encryption Standard.

* Pencipta: IBM
* Kategori: Block Cipher
* Ukuran block (bit): 64
* Ukuran kunci (bit): 56 (+8 parity bits)
* Jumlah round: 16
* Struktur: Balanced Feistel Network
* Diturunkan dari: 
    - [Lucifer](../Lucifer)
* Penerus: 
    - Triple DES
    - G-DES
    - DES-X
    - LOKI89
    - [ICE](../ICE)
