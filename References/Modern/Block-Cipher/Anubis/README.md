# Crypto Reference

## Anubis Cipher

* Pencipta:
    - Vincent Rijmen
    - Paulo S. L. M. Bareto
* Kategori: Block Cipher
* Ukuran block (bit): 128
* Ukuran kunci (bit): 128 / 160 / 192 / 224 / 256 / 288 / 320
* Jumlah round: 12 / 13 / 14 / 15 / 16 / 17 / 18
* Struktur: Substitution-Permutation Network
* Diturunkan dari: 
    - [Rijndael](../Rijndael)
