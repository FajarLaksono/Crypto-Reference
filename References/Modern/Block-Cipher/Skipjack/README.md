# Crypto Reference

## Skipjack Cipher

* Pencipta: NSA (National Security Agency)
* Kategori: Block Cipher
* Ukuran block (bit): 64
* Ukuran kunci (bit): 80
* Jumlah round: 32
* Struktur: Unbalanced Feistel Network
* Diturunkan dari: N/A
* Penerus: N/A

NIST merekomendasikan untuk tidak menggunakan Skipjack setelah 2010.
