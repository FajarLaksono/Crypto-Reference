# Crypto Reference

## Q Cipher

* Pencipta: Leslie McBride
* Kategori: Block Cipher
* Ukuran block (bit): 128 
* Ukuran kunci (bit): 128 / 192 / 256
* Jumlah round:
    - 8 (untuk ukuran kunci 128-bit)
    - 9 (untuk ukuran kunci 192 / 256 bit)
* Struktur: Substitution-Permutation Network
* Diturunkan dari: 
    - [Rijndael](../Rijndael)
    - [Serpent](../Serpent)
* Penerus: N/A
