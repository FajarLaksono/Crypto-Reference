# Crypto Reference

## XXTEA (Corrected Block TEA) Cipher

* Pencipta:
    - David Wheeker
    - Roger Needham
* Kategori: Block Cipher
* Ukuran block (bit): >= 64
* Ukuran kunci (bit): 128
* Jumlah round: (bergantung terhadap ukuran block)
    - ~52+6*words (6-32 full cycle)
* Struktur: Unbalanced Feistel Network
