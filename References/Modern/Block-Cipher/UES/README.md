# Crypto Reference

## UES Cipher

Universal Encryption Standard.

* Pencipta:
    - Helena Handschuh
    - Serge Vaudenay
* Kategori: Block Cipher
* Ukuran block (bit): 128
* Ukuran kunci (bit): 128 / 192 / 256
* Jumlah round: 48
* Struktur: Feistel Network
* Diturunkan dari: 
    - [DEAL](../DEAL)
    - Triple DES
