## IDEA NXT Cipher

Awalnya dikenal sebagai sandi FOX.

* Pencipta:
    - Pascal Junod
    - Serge Vaudenay
* Kategori: Block Cipher
* Ukuran block (bit): 64 / 128
* Ukuran kunci (bit): 0-256
* Jumlah round: 16
* Struktur: Lai-Massey Scheme
