## KeeLoq Cipher

KeeLoq adalah block cipher yang khusus diimplementasikan pada perangkat terdedikasi (proprietary hardware-dedicated block cipher).

* Pencipta:
    - Gideon Kuhn (algoritma)
    - Willem Smit (implementasi chip)
* Kategori: Block Cipher
* Ukuran block (bit): 32
* Ukuran kunci (bit): 64
* Jumlah round: 528
