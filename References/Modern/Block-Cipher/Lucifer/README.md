## Lucifer Cipher

* Pencipta: Horst Feistel
* Kategori: Block Cipher
* Ukuran block (bit): 32 / 48 / 128
* Ukuran kunci (bit): 48 / 64 / 128
* Jumlah round: 16
* Struktur: 
    - Substitution-Permutation Network
    - Feistel Network

* Penerus:
