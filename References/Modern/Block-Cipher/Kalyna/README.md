# Crypto Reference

## Kalyna Cipher

Sandi yang digunakan sebagai standard di Ukraina sejak 2015 (DSTU 7624:2014).

* Pencipta:
* Kategori: Block Cipher
* Ukuran block (bit): 128 / 256 / 512
* Ukuran kunci (bit): 128 / 256 / 512
* Jumlah round: 
    - 10 (jika kunci 128)
    - 14 (jika kunci 256)
    - 18 (jika kunci 512)
* Struktur: Substitution-Permutation Network
* Diturunkan dari: 
    - [Rijndael](../Rijndael)
* Penerus: N/A
