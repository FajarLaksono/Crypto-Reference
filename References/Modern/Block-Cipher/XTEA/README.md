## XTEA (eXtended Tiny Encryption Algorithm) Cipher

* Pencipta:
    - David Wheeker
    - Roger Needham
* Kategori: Block Cipher
* Ukuran block (bit): 64
* Ukuran kunci (bit): 128
* Jumlah round: 32
* Struktur: Lai-Massey Scheme
