# Crypto Reference

## MISTY1 Cipher

* Pencipta: Mitsubishi Electric (Mitsuru Matsui dkk)
* Kategori: Block Cipher
* Ukuran block: 64
* Ukuran kunci: 128
* Jumlah round: 4*N (direkomendasikan 8)
* Struktur: Nested Feistel Network
* Diturunkan dari: N/A
* Penerus: 
    - [Camellia](../Camellia)
    - MISTY2
    - [KASUMI](../KASUMI)
* Certification:
    - CRYPTREC
    - NESSIE
