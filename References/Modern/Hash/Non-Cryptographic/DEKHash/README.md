# Crypto Reference

## DEK Hash

Hash function yang diciptakan oleh Donald E. Knuth dalam buku "The Art of Computer Programming: Volume 3" pada topik "sorting and search" chapter 6.4.

* Kategori: Non-Cryptographic Hash
* Ukuran block:  8-bit
* Ukuran state: 32-bit
* Ukuran hash:  32-bit
* IV: panjang string
