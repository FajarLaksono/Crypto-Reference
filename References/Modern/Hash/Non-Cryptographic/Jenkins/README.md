# Crypto Reference

## Jenkins Hash

Hash function yang diciptakan oleh Bob Jenkins dengan beberapa varian.

* Kategori: Non-Cryptographic Hash
* Ukuran block:  8-bit
* Ukuran state: 32-bit
* Ukuran hash:  32-bit
* IV: 0
